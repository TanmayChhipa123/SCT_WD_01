RESPONSIVE
LANDING PAGE

Welcome to my SkillCraft Task 01 project. This is a clean and responsive landing page with interactive navigation.

Features

This page is built with HTML, CSS, and JavaScript and includes a fixed navbar, scroll effects, hover transitions, and responsive layout. The navbar changes style when scrolling, and each section is clearly structured and easy to navigate.

About This Project

This landing page is part of Task 01 from SkillCraft Technology. The goal was to create a responsive design with a fixed-position navigation bar that changes style when hovered or when scrolling down the page. This project helped me practice DOM manipulation, styling, and responsive layout techniques.
